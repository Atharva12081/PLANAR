# PLANAR

PLANAR (Planetary Latent Analysis & Representation) is an end-to-end framework for:

- unsupervised clustering of ALMA disk images in learned latent space,
- reconstruction-driven representation learning with convolutional autoencoders,
- 1D-CNN transit detection on synthetic and stress-tested light curves.

See [`README.md`](../README.md) for quickstart and benchmark summary.
